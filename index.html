FAQ

Common Questions

Does ClutchReframe Clips provide any competitive advantage?

No. ClutchReframe Clips is a recording and clipping tool only. It does not display any real-time game information during gameplay, does not track abilities or cooldowns, and does not provide any data that could affect competitive play. The in-game overlay only shows recording status reminders.

What data does the app collect?

Only game events (kills, deaths, assists) via Overwolf's official Game Event Provider for highlight detection, video recordings via the Overwolf Replays API, and your local app preferences. All data is stored locally on your computer. We do not collect personal information or transmit any data to external servers.

Does the app read game memory or modify game files?

No. ClutchReframe Clips only uses Overwolf's official Game Event Provider (GEP) API and the Replays API. It does not access game memory, modify game files, or inject code into the game process.

Where are my clips stored?

All clips and settings are stored locally on your computer. Nothing is uploaded to any server. You can delete clips at any time from within the app, and uninstalling removes all associated data.

Is ClutchReframe Clips affiliated with Riot Games?

No. ClutchReframe Clips is an independent application built on the Overwolf platform. It is not endorsed by or affiliated with Riot Games in any way.

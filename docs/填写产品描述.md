● 完整版英文：
ClutchReframe Clips is an automatic highlight capture and intelligent clipping tool for League of Legends, built on the Overwolf platform.

Core Features:
1. Smart Highlight Capture — Captures kill, death, and assist events through Overwolf Game Event Provider (GEP), combined with visual analysis to determine battle status. Automatically backtracks to find the true start of combat and intelligently identifies when the fight ends, capturing complete highlight moments rather than fixed-length clips.
2. Vertical Smart Crop (9:16) — After the game, a lightweight local vision algorithm analyzes each highlight, tracks the action, and crops it into a perfect 9:16 vertical video, ensuring key combat details stay in frame. One click, ready for social media platforms.
3. Fully Local Processing — Every video, every clip, every setting, stored only on your own machine. No servers. No uploads. Your gameplay stays yours.

This application does not use any Riot Games API endpoints. All game event data comes exclusively from Overwolf's Game Event Provider (GEP), accessed through Overwolf's officially licensed partnership with Riot Games. The app does not provide any competitive advantage and does not track enemy cooldowns or abilities.

We are registering to comply with Riot's third-party application policies for distribution through the Overwolf app store.

中文翻译：

ClutchReframe Clips 是基于 Overwolf 平台的英雄联盟自动精彩时刻捕捉与智能剪辑工具。

核心功能：
1. 智能精彩捕捉 — 通过 Overwolf 游戏事件提供器 (GEP) 捕捉击杀、死亡和助攻事件，结合视觉分析判断战斗状态。自动回溯找到战斗的真正起点，并智能识别战斗结束时刻，捕捉完整的精彩时刻，而非固定时长的片段。
2. 竖屏智能裁剪 (9:16) — 赛后，本地轻量级视觉算法逐一分析每个精彩片段，追踪动作焦点，裁剪为完美的 9:16 竖屏视频，确保关键战斗细节始终在画面内。一键完成，即可发布至社交媒体平台。
3. 完全本地处理 — 每段视频、每个片段、每项设置，都只存储在你自己的电脑上。无服务器，无上传。你的游戏画面属于你自己。

本应用不使用任何 Riot Games API 端点。所有游戏事件数据仅来自 Overwolf 的游戏事件提供器 (GEP)，通过 Overwolf 与 Riot Games 的官方授权合作关系获取。应用不提供任何竞争优势，不追踪敌方冷却或技能。

我们注册是为了遵守 Riot 第三方应用政策，以便通过 Overwolf 应用商店分发。
